What This Does

Adds a new field to store WeChat ID or Phone Number for supplier contacts.

SQL to Execute in Supabase

-- Add wechat_phone column to suppliers table ALTER TABLE suppliers ADD COLUMN IF NOT EXISTS wechat_phone VARCHAR(100); -- Add comment to clarify the column purpose COMMENT ON COLUMN suppliers.wechat_phone IS 'WeChat ID or Phone Number for supplier contact'; -- Check the column was added SELECT column_name, data_type, is_nullable FROM information_schema.columns WHERE table_name = 'suppliers' AND column_name = 'wechat_phone';

Instructions:

  1. Copy the SQL above
  2. Go to your Supabase Dashboard
  3. Navigate to SQL Editor
  4. Paste and run the SQL
  5. The column will be added and you'll see confirmation
